# Milon Veterinary

ভেটেরিনারি ফার্মেসি ও ফিড দোকানের জন্য একটি ডেমো ম্যানেজমেন্ট সফটওয়্যার টেমপ্লেট।

## টেক স্ট্যাক
- PHP
- Bootstrap 5
- jQuery
- MySQL

## ফিচার
- লগইন
- বিজনেস ইনফো
- প্রোফাইল আপডেট
- আয়-ব্যয়
- দায়-পরিসম্পদ
- ইনভেন্টরি
- রিপোর্ট
- মেমো ম্যানেজমেন্ট (ভিউ/প্রিন্ট/ডিলিট)
- নতুন মেমো (পণ্য বিক্রয়)
- কাস্টমার ম্যানেজমেন্ট
- ডিউ ম্যানেজমেন্ট
- খরচ ম্যানেজমেন্ট + খরচের খাত
- সাপ্লায়ার ম্যানেজমেন্ট
- ক্যাটেগরি ম্যানেজমেন্ট
- পণ্য ক্রয় (নগদ/আংশিক/বাকি)
- লাইট/ডার্ক মোড

## রান করার নিয়ম
1. PHP সার্ভার চালান:
   ```bash
   php -S localhost:8000
   ```
2. ব্রাউজ করুন: `http://localhost:8000`

## ডাটাবেস
`database.sql` ফাইলটি চালিয়ে ডাটাবেস তৈরি করুন এবং `db.php` ফাইলে কানেকশন ক্রেডেনশিয়াল আপডেট করুন।

প্রথমবার লগইনের জন্য `users` টেবিলে একটি ইউজার যোগ করতে হবে। উদাহরণ:
```sql
INSERT INTO users (name, username, password_hash, role)
VALUES ('Milon Admin', 'admin', '$2y$10$ZfWGYLfi9aYxHmyo7LIsIu0V4Um5mO4c4l9kH6cYg8e2PH6mEwO4y', 'Owner');
```
উপরের পাসওয়ার্ড হ্যাশটি `admin123` এর জন্য।

## নোট
এই টেমপ্লেটটি স্টার্টার হিসেবে তৈরি। প্রয়োজনে ফর্ম সাবমিশন, ভ্যালিডেশন ও ডায়নামিক ডাটা যুক্ত করতে হবে।
